Project Overview

Riyora is a portfolio-ready, educational e-commerce demonstration showcasing secure web design, SQLite integrations, and modern front-end concepts.

Security Features
  • 🔒 CSRF & Rate Limiting
  • 🛡 Session Expire Middlewares
  • 🔑 Bcrypt Password Hashing
  • 🔌 Google OAuth 2.0 Auth
Technology Stack
  • Core: Python Flask
  • Database: SQLite3
  • Frontend: CSS3 / Vanilla JS
  • Deploy: Gunicorn / Docker
